# cube2D-3D grahics project
JS Project: html canvas 2D, 3D cube practice with genAI, Github Copilot. 
That is experimental project with genAI with the different LLM.
The desired steps and functionalities are generated by the different LLM.
Programmer just checks the generated codes part and validate them.

## Steps of the cube project
- step1: Creating coordinate-system
- step2: coordinate-system with drawing a 2D cube with the rotatation, scala
- step3: Drawing more 2D cubes (count: n), extra feature moving.
- step4: Drawing 3D cubes and insert them with the preserved features (rotation, scale, moving)

## Supported languages
- Default lang: vanillaJs + html canvas
- Libraires:
    - BabylonJs
    - Vite build tool
- It will be written by Python: 
    - v3.x
    - graphics lib
